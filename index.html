How to Play 📖

Objective:
Fill the 9×9 grid so that each row, column, and 3×3 box contains the numbers 1-9 without repetition.

Rules:
• Each row must contain numbers 1-9
• Each column must contain numbers 1-9
• Each 3×3 box must contain numbers 1-9
• No number can repeat in any row, column, or box

Controls:
• Tap a cell to select it
• Use the number pad to fill cells
• Use hints when you're stuck (3 per game)
• Tap ⌫ to delete a number
• Submit when you're done

Tips:
• Start with rows/columns that have many numbers
• Look for numbers that can only go in one place
• Take your time and breathe 🧘

About Zenku ⛩️

What is Sudoku?
Sudoku is a logic-based number puzzle that originated in Japan. It challenges your mind and improves concentration, logic, and problem-solving skills.

Why Play Zenku?
Find your stillness in the grid.

Zenku (Zen + Sudoku) beautifully fuses the classic logical challenge of 9×9 Sudoku with the tranquil, minimalist beauty of Japanese Zen culture. This is not just a puzzle; it is a path to clarity.

Every element is designed to promote calm and composure. Immerse yourself in a peaceful aesthetic featuring muted, natural colours and soothing, ambient soundscapes. There are no harsh timers or jarring alerts—just the satisfying, quiet moment of true concentration.

Solve, meditate, and achieve Satori (sudden enlightenment) as you complete each grid. If you seek a focused retreat and an experience that is gentle on the eyes and good for the soul, Zenku is your sanctuary. 🧘‍♂️
